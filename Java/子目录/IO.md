# IO

- io分为输入和输出，也可以分为字节流和字符流
  - 字节流：以8位（1byte，8个bit）作为一个数据单元，数据流中最小的数据单元是字节
  - 字符流：以16位（1char，2byte，16bit）作为一个数据单元，在Java中一个字符是两个字节
